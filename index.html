Про мене

Студент кафедри інформаційної безпеки, відомий як Alex Melan (A.K.A. Alex_Melan). З шкільних років захоплююсь програмуванням, комп'ютерним залізом, інформаційною безпекою, написанням модів та плагінів для Minecraft, а також ботів для Telegram/Discord. Володію власним VPN, досліджую криптовалюти, децентралізовані мережі (i2p, Yggdrasil Network), нейронні мережі та адміністрування серверів.

Програмую в своє задоволення. У житті за власним бажанням вивчаю Python, Java, Lua, Rust та OpenGL. За університетом ще стикався з C++ та C#. Вайбкодив кілька проектів на Go.

Мій шлях

🏫 Шкільні роки

Почав писати ботів для Telegram та Discord. Після вивчення створення сайтів зацікавився безпекою шкільного сайту. Досліджував методи атаки на WiFi мережі та працював з інструментами типу Pixie WPS, Reaver, Bully, Hashcat та JohnTheRipper.

🎓 Університет: 1-2 курс

Писав програми для автоматизації розрахунків у лабораторних роботах. На другому курсі знайшов першу уразливість в електронній зачітці університету.

🔥 Університет: 3 курс

Розвив попередні знахідки в повний аналіз безпеки електронної зачітки. Виявив критичні уразливості: застарілі версії бібліотек, вразлива стара ОС з 2014 року, відсутність шифрування, відсутність авторизації для запитів до БД, відкриті API тощо. На базі знайдених даних створив інструмент, який спрощує роботу з зачіткою та надає більш обширний функціонал.

🚀 Сьогодення

Продовжую вивчати нові технології, працювати над проектами в криптовалютах, нейронних мережах, адмініструванні серверів. Досліджую децентралізовані мережі, пишу моди та плагіни для Minecraft, розгортаю локальні LLM та експериментую з різними технологіями.

Напрямки діяльності

🔐

Інформаційна безпека

Пентестинг та аналіз вразливостей. Досвід з WiFi атаками (Pixie WPS, Reaver, Bully) та інструментами злому (Hashcat, JohnTheRipper). У школі тестував безпеку шкільного сайту. В університеті виявив критичні уразливості в електронній зачітці: застарілі бібліотеки, вразлива ОС з 2014 року, відсутність шифрування та авторизації для запитів до БД, відкриті API.

🤖

Боти та автоматизація

Ще зі школи почав писати ботів для Telegram та Discord. В університеті створюю програми для автоматизації розрахунків у лабораторних роботах. На базі знайдених даних про уразливості зачітки створив інструмент, який спрощує роботу з нею та надає більш обширний функціонал.

🎮

Minecraft Development

Пишу моди та плагіни для Minecraft. Тримаю кілька власних серверів для гри з друзями. Адмініструю, обслуговую, пишу плагіни за запитом та виступаю адміном, модератором, власником. Все в одному.

🧠

Машинне навчання

Дуже цікавлюсь нейронними мережами. Зокрема GAN (Lightweight-GAN та StyleGAN). Пишу свої простенькі мережі, намагаюсь навчити готові архітектури на своїх даних та вивчаю різні техніки. Також цікавить синтез голосу за допомогою нейронних мереж. Розгортаю локально у себе LLM через Ollama.

₿

Криптовалюти

Вивчаю криптовалюти та реалізував кілька своїх проектів, застосовуючи кастомні самописні бібліотеки. Писав з нуля бібліотеку для HD адрес та кастомізував її під свої напрацювання для написання власної криптовалюти.

🌐

Децентралізовані мережі та VPN

Цікавлюсь децентралізованими одноранговими мережами типу i2p та Yggdrasil Network. Володію кількома серверами, що надають послуги VPN з різними протоколами (кожен сервак під свої задачі: OpenVPN та XRAY + VLESS + Reality).
